# 秋招面试-京东

## 笔试

题型：
- 30道选择题
- 2道算法题


算法题：

1. 小明要在n天吃饭，他会做m道菜。但是相邻两天做的菜的编号要满足(|ai-aj|>=k)，求有多少种序列的做法（1<=n,m<=1000, 0<=k<m)
2. 在(n,m)(1<=n,m<=2000)的地图分为可通行和障碍物两种，机器人需要从起始点走到终点。机器人有点问题：上下移动后，下次只能左右移动；左右移动后，下次只能上下移动。求起点到终点需要的最小步数，到不了输出-1


```java
// 83%；91%（第二题RTE）
import java.io.*;
import java.util.*;

public class Main {
    
    static int m;
    static int n;
    
    static int targetX;
    static int targetY;
    
    public static void main(String[] args) throws IOException {
        n = nextInt();
        m = nextInt();
        
        int x1 = nextInt() - 1;
        int y1 = nextInt() - 1;
        targetX = nextInt() - 1;
        targetY = nextInt() - 1;
        
        boolean[][] maps = new boolean[n][m];
        for (int i = 0; i < n; ++i) {
            String s = next();
            for (int j = 0; j < m; ++j) {
                if (s.charAt(j) == '.') {
                    maps[i][j] = true;
                }
            }
        }
        boolean[][][] visited = new boolean[n][m][2];
        
        // 需要从终点一层一层的往外去找，直到确定步数
        // 队列 targetX, targetY, direction
        Deque<int[]> deque = new ArrayDeque<>();
        deque.offer(new int[]{targetX, targetY, 0});
        deque.offer(new int[]{targetX, targetY, 1});
        
        // 初始点已访问
        visited[targetX][targetY][0] = true;
        visited[targetX][targetY][1] = true;
        
        int ans = Integer.MAX_VALUE;
        
        int step = 0;
        boolean findAnswer = false;
        while (!deque.isEmpty() && !findAnswer) {
            int n = deque.size();
            ++step;
            for (int i = 0; i < n; ++i) {
                int[] elements = deque.poll();
                int x = elements[0];
                int y = elements[1];
                if (x == x1 && y == y1) {
                    ans = step - 1;
                    findAnswer = true;
                    break;
                }
                int direction = elements[2];
                if (direction == 0) {
                    if (inBound(x, y - 1) && !visited[x][y - 1][1] && maps[x][y - 1]) {
                        visited[x][y - 1][1] = true;
                        deque.offer(new int[]{x, y - 1, 1});
                    }
                    if (inBound(x, y + 1) && !visited[x][y + 1][1] && maps[x][y + 1]) {
                        visited[x][y + 1][1] = true;
                        deque.offer(new int[]{x, y + 1, 1});
                    }
                } else {
                   if (inBound(x - 1, y) && !visited[x - 1][y][0] && maps[x - 1][y]) {
                        visited[x - 1][y][0] = true;
                        deque.offer(new int[]{x - 1, y, 0});
                    }
                    if (inBound(x + 1, y) && !visited[x + 1][y][0] && maps[x + 1][y]) {
                        visited[x + 1][y][0] = true;
                        deque.offer(new int[]{x + 1, y, 0});
                    }
                }
            }
        }
        
        
        if (ans == Integer.MAX_VALUE) {
            System.out.print(-1);
        } else {
            System.out.print(ans);
        }
        
        
    }
    
    public static boolean inBound(int x, int y) {
        return x >= 0 && x < n && y >= 0 && y < m;
    }
}
```
